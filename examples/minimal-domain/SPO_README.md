# writing_judgment

Editorial writing judgment — diagnose whether content has a real argument, a cognitive hook, and evidence density. Not copy editing.

## Where it comes from

Derived from editorial expertise in diagnosing structural vs surface-level writing problems across hundreds of content reviews.

## Where it applies

- User asks to review content
- User asks to improve writing
- User submits draft for feedback

## How it is verified

- 0 eval cases (0 rated)
- 1 locked axioms with applies_when / does_not_apply_when / failure_risk
- 1 self-check questions
- 1 misunderstanding patterns

## When it does NOT apply

- User explicitly asks for grammar check only
- User asks for translation
- User asks for formatting

## Top Axioms

- **Most writing problems are structural and argument-level, not language-level.**
  - Failure risk: May cause the agent to over-diagnose structural problems in content that genuinely only needs language polishing.

## Top Misunderstandings

- WRONG: Good writing means clear sentences and proper grammar.
  CORRECT: Good writing means having a clear argument supported by specific evidence, with a hook that captures attention in the first three sentences.
